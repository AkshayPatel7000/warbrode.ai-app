# **Home Screen – Feature Description (For UI Development)**

The Home Screen serves as the main dashboard of the Smart Wardrobe AI app. It provides an overview of the user’s day, recommended outfit, quick wardrobe stats, and essential shortcuts. Each section is modular and must be implemented as reusable UI components.

---

## **1. Header Section**

**Elements**

* **User Avatar** (top-left)

  * Displays the logged-in user’s profile picture.
* **App Title: “Smart Wardrobe”**

  * Bold, primary title.
* **Greeting Line: “Good Morning!”**

  * Time-based dynamic greeting (Morning/Afternoon/Evening).
* **Notification Icon** (top-right)

  * Shows unread alerts (optional badge).

**Design**

* Placed over a **pastel lime gradient background**.
* Full-width, rounded bottom corners.

---

## **2. Today’s Outfit Card**

**Purpose**
Shows the AI-generated outfit for the day with preview and style context.

**Elements**

* **Outfit Image Placeholder**

  * Flat-lay outfit image generated by backend (preview URL).
* **AI Recommendation Text**

  * “AI picked this for you”
* **Outfit Title**

  * Example: “Casual Day Out”
* **CTA Button: “View Outfit”**

  * Leads to outfit detail screen.

**Design**

* Large card with rounded corners and white background.
* Soft shadow for elevation.
* 4:3 image ratio at top.

---

## **3. Weather Card**

**Purpose**
Shows today’s weather and whether it influenced outfit selection.

**Elements**

* **Weather Icon + Temperature + Condition**

  * Example: “21°C & Sunny”
* **Subtitle**

  * “Weather applied to outfit”
* **Arrow icon**

  * May navigate to detailed weather preferences/settings.

**Design**

* Small rounded tile (card-style).
* White background, subtle shadow.

---

## **4. Wardrobe Stats Section**

**Purpose**
Shows quick counts of wardrobe items to indicate wardrobe size & variety.

**Three Stat Cards**

* **Tops**

  * Icon + count (e.g., 124)
* **Bottoms**

  * Icon + count (e.g., 58)
* **Shoes**

  * Icon + count (e.g., 32)

**Design**

* Three equal-width tiles
* Rounded corners, white background
* Centered icon on top, count below
* Soft shadows for elevation

---

## **5. Quick Action Buttons**

**Purpose**
One-tap access to key features.

**Buttons**

* **Upload**

  * Opens upload/bottom sheet to add clothing.
* **Generate**

  * Triggers outfit generation page.
* **History**

  * Opens outfit history timeline.

**Design**

* Rounded pill buttons with icons.
* Placed in a row beneath stats.

---

## **6. Bottom Navigation Bar**

**Tabs**

* Home (active)

* Generate
* History
* Profile

**Design**

* Floating pill-shaped container.
* Soft glassmorphism blur background.
* Active icon highlighted with lime background.
